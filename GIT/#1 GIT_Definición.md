# GIT

Es un sistema de control de versiones distribuido,
permitiendo que cada desarrollador tenga una copia completa del proyecto en su equipo,
en conjunto con su historial.

- No requiere un servidor núlcleo
- Eficiente y bien obtimizado
- Altamente seguro y muy fiable

### Flexibilidad

El lenguaje permite realizar modificaciones sobre un archivo por varias personas a la vez,
usando un método de ramas de desarrollo para que los cambios de cada localhost no interfiera entre sí

## Ramas de Trabajo

1. Directorio
   - Editor
   - Ruta
   - index.modiffied
2. Área de Trabajo
   - Servicio de Staging
   - Sync Cloud
   - index.station
3. Repositorio
   - GitHub
   - GitLab
   - index.commitel

**NOTA:** GitBash es la Shell de Git.
