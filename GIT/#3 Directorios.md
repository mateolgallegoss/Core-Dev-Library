# Directorios

## Navegación entre directorios
### Acceso a directorios
Acceso a un disco
~~~
$ cd (letra):
~~~
Acceso a un directorio
~~~
$ cd (directorio)
~~~
### Navegación entre directorios
Retroceso entre directorios
~~~
$ cd ../
~~~
Usado una vez se inicializo .git
~~~
$ cd ..
~~~
Comprobar las carpetas existentes en el directorio actual
~~~
$ ls 
~~~
Brinda información sobre el área de trabajo y preparación
~~~
$ git status
~~~
### Manipulación de directorios
Crear un directorio 
~~~
$ mkdir (nombre)
~~~
Remover un directorio
~~~
$ rmdir (nombre):
~~~
Habilitar acceso a directorios ocultos
~~~
$ ls -a
~~~
